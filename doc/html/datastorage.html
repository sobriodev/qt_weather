    Detailed Description

    The DataStorage class stores all readouts data and manages serializing and deserializing them

    Member Function Documentation

    DataStorage::DataStorage(int validM = DefaultValidMinutes)

    The DataStorage class constructor. The validM parameter is used to set data saving interval (in minutes). Correct interval configurations are:

    • validM >= 10 AND validM <= 5 Save data with [currentTime] % [validM] interval. For example setting validM to 20 will cause data saving when 00, 20, 40 minutes are reached
    • validM = 60 Save data when next fully hour is reached

    void DataStorage::append(const SensorData &readout)

    Save data readout using readout argument. Only valid readouts are saved

    void DataStorage::deleteOldReadouts(int days, const QTime &timeSub)

    Delete old readouts from storage. The cricital delete date is built from days and timeSub arguments. For example setting days argument to 5 and timeSub to 1 hour, 31 minutes and 5 seconds deletes all readouts older or equal 5 days, 1 hour, 31 minutes and 5 seconds from now

    void DataStorage::deserialize(const QJsonObject &input)

    Deserialize Storage from JSON object where input stores serialized JSON data

    void DataStorage::serialize(QJsonObject &output) const

    Serialize Storage data. The output argument is an output JSON object
